# 极其简陋的蹭课脚本v0.01
---
自己想选择的课炒鸡火爆选不上？［滑稽］
那就试试这个腊鸡脚本吧！
## 用法
［目前还不具备查询，自动登录功能，需要自行查询并复制链接中课程参数］
将需要蹲点的课填入course中
```
dundiancengke.init({
		course : "kch=258321&bjh=002297B4",
		id : "",
		pwd : ""
	})
	//人机交互kch=267031&bjh=003466B2
	//应用密码学kch=262053&bjh=002562B1
	//动态网站kch=262047&bjh=003432B1
```
登入选课系统，将脚本复制到浏览器检查器中运行。
